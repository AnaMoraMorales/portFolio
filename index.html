VIVEROS MORA

El proyecto Viveros Mora se centra en la creación de una aplicación móvil para la compra y administración de plantas. La aplicación está diseñada para ofrecer información detallada y personalizada sobre las plantas disponibles, categorizadas según el perfil del cliente (interior, exterior, de fácil cuidado, entre otros). Utilizando tecnologías como Ionic y Firebase, la app permite a los usuarios registrarse, autenticarse, comprar productos, y gestionar su perfil. La base de datos no relacional Firebase se utiliza para almacenar información sobre usuarios, productos, y pedidos, además de gestionar autenticaciones y recomendaciones personalizadas.

Entre las funcionalidades implementadas destacan:

  • Registro y autenticación de usuarios: Utilizando Google o email y contraseña.
  • Visualización de catálogo: Los usuarios pueden navegar por el catálogo de productos, filtrando por categorías como plantas de interior o exterior.
  • Recomendaciones personalizadas: La aplicación permite a los usuarios realizar un test para recibir recomendaciones personalizadas basadas en sus preferencias.
  • Gestión de pedidos: Los usuarios pueden realizar pedidos y hacer seguimiento del estado de estos (en preparación, enviado, en reparto, entregado).
  • Framework y herramientas: El desarrollo fue realizado con Ionic y Firebase en el entorno Visual Studio Code, permitiendo un diseño ágil y adaptado a móviles.
